## 项目配置
### 环境变量
- `VITE_PORT`: 开发服务器端口（默认3001）
- `VITE_OPEN`: 是否自动打开浏览器
- `VITE_USE_COMPRESSION`: 是否启用 gzip 压缩
- `VITE_USE_CONSOLE`: 是否启用控制台日志
